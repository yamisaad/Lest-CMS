/*********************************Tutoriels  du CMS*******************************************/
-pour avoir le psd du header et footer "app/theme/asterionv9/template/img/psd/"
- configurer soigneusement la page "config/config.php"
-pour les liens starpass regarder le fichier starpass.txt
-mettez les sql "sql/v0.1" sur votre table d'emulateur
*pour ajouter un item boutique veuillez mettre l'id de l'item : 
-mettez le swf de l'item dans "app/theme/boutique/swf/"
-ajouter la ligne sql de l'item sur votre table item_template
